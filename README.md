# CS4337 Project 1 – Prefix Calculator

This project implements a prefix-notation expression calculator written in Racket.

## Files
main.rkt – calculator implementation  
devlog.md – development log of the project  

## Running the Program

Interactive mode:
racket main.rkt

Batch mode:
racket main.rkt -b

## Features

- Prefix expression evaluation
- Operators supported:
  - +
  - *
  - /
  - unary -
- History references using `$n`
- Error handling for invalid expressions
